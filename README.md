# 🛒 Amazon E-Commerce Database Management System

This repository contains the initial documentation for the **Amazon E-Commerce Database Management System** project. The project focuses on designing a relational database for an Amazon-like online shopping platform by analyzing business requirements and identifying database entities, attributes, and relationships.

## 📂 Files Included

- 📄 Requirement Analysis.pdf
- 📄 Entity Analysis.pdf

## 📖 Project Description

The project aims to design a structured and efficient database for an e-commerce platform similar to Amazon. It covers the essential database planning phase, including system requirements and entity analysis before implementation.

## 🎯 Objective

To design a scalable and normalized database structure for managing customers, sellers, products, orders, payments, inventory, shipping, and reviews in an Amazon-style e-commerce system.
